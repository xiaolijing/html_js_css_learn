border-radius实现圆角边

实现类型 实现效果 实现说明
全面圆
高度和宽度一致，border-radius的值大于高宽一半
半圆
高度是宽度一半
border-radius: 50px 0 0 50px;
按照左上，右上，右下，左下的顺时针方向设置

阴影框

实现类型 实现效果 实现说明
外阴影
阴影参数：
值 描述
X轴偏移量 必须；水平阴影位置，允许负值
必须；抽纸阴影位置，允许负值
阴影模糊半径 可选，模糊距离，值越大，阴影模糊边界越大
阴影扩展半径 可选，阴影尺寸，值可以为负数，阴影的效果是模糊半径+扩展半径，扩展半径不会模糊显示
阴影颜色 可选，阴影颜色，默认为黑色
投影方式 可选（设置inset时为内部阴影方式，如果忽略为外阴影方式）
内阴影
box-shadow: 4px 4px 6px #666 inset;
多阴影
多阴影属性，以“,”进行多定义
box-shadow: 4px 2px 6px #f00, -4px -2px 6px #000, 0px 0px 12px 5px #33CC00 inset;
